About This Project

Ecommerce Recommendation System is a Final Year B.Tech project developed by Shaik Nagoor Sharief (KL University).

This platform helps users discover products using intelligent machine learning models.

It leverages Content-Based Filtering and Natural Language Processing (NLP) to recommend similar products based on name, brand, tags, and reviews.

Technologies Used: Flask, Pandas, Scikit-learn, MySQL, HTML, CSS, Bootstrap.
